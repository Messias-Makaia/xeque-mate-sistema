# xeque-mate-sistema
O Xeque-Mate sistema contabilístico é um sistema contábil básico para a microempresa Fármacos Xeque-Mate. O principal objectivo do sistema definido para o escopo inicial do projecto é a informatização do processo contabilístico da Xeque-Mate, permitindo o registo de lançamentos e a geração de relatórios financeiros.
